Typography guidelines

Change the size, weight and other font properties of the text by adding the following classes



SIZE


Class Font-size Example
is-size-1 3rem Example
is-size-2 2.5rem Example
is-size-3 2rem Example
is-size-4 1.5rem Example
is-size-5 1.25rem Example
is-size-6 1rem Example
is-size-7 0.75rem Example

ALIGNMENT


Class Alignment Example
has-text-centered Makes the text centered Example
has-text-justified Makes the text justified Example
has-text-left Makes the text left Example
has-text-right Makes the text right Example

TEXT TRANSFORMATION


Class Transformation Example
is-capitalized Transforms the first character of each word to uppercase Example
is-lowercase Transforms all characters to lowercase Example
is-uppercase Transforms the all characters to uppercase Example
is-italic Transforms all characters to italic Example
is-underlined Underlines the text Example

TEXT WEIGHT


Class Weight Example
has-text-weight-light Transforms text weight to light Example
has-text-weight-normal Transforms text weight to normal Example
has-text-weight-medium Transforms text weight to medium Example
has-text-weight-semibold Transforms text weight to semi-bold Example
has-text-weight-bold Transforms text weight to bold Example
